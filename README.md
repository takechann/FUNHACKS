## プロダクト名
Coord Studio
##製品概要
着る服を自動で選んでくれるWEBアプリ
1.　ユーザがカメラで自分が持っている服の写真をインポート
2.　アプリが服の色や今日の気温を元にコーディネートを提示
##服選びに対する人の不満
1.　服選びの手間をなくしたい
2.　その日の気温に合った服装がわからない
3.　選ぶのが疲れる[1][2] 日本の女性の半分が服選びにストレスを感じている
##効果
1.　服選びの時間短縮
2.　気温に適した服装の着用が可能になる[3]
3.　選択・決断の負担を解消される
